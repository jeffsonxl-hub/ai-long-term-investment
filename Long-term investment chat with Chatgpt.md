# 九、Risk Agent

这是我认为很多个人投资者最缺失的一环。

专门负责找"不能买"的理由，而不是寻找买入理由。

风险类型包括：

### 财务风险

- 应收账款异常增长
- 存货持续积压
- 商誉占净资产比例过高
- 连续多年自由现金流为负
- 审计意见异常

### 治理风险

- 大股东高比例股权质押
- 频繁关联交易
- 管理层频繁更换
- 重大诉讼或处罚

### 行业风险

- 行业景气度下行
- 技术路线被替代
- 政策发生重大变化

### 市场风险

- 波动率异常
- 流动性不足
- 融资融券风险

Risk Agent 更像一个"反对者"，任何高风险信号都会降低综合评分。

## ② Portfolio Manager Agent（组合管理）

这是最终与投资者直接交互的 Agent。

- 评估组合风险（Beta、最大回撤、波动率）
- 提供调仓建议
- 跟踪投资假设是否发生变化

## 我建议的下一步

如果你的目标是打造一个**真正接近基金公司研究体系的个人 AI 投资平台**，我建议不要止步于指标列表，而是进一步设计完整的数据与 Agent 体系，包括：
